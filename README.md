# 🎂 Age Calculator

A simple and clean Age Calculator built using HTML, CSS, and JavaScript.

## 🚀 Features

- Calculate exact age in:
  - Years
  - Months
  - Days
- Shows:
  - Total months
  - Total weeks
  - Total days
  - Total hours
- Displays next birthday countdown 🎉
- Smooth UI with background video

## 🛠️ Technologies Used

- HTML
- CSS
- JavaScript
